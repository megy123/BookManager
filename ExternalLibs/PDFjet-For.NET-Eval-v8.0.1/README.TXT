Thank you for evaluating PDFjet for .NET v8.0.1


To rebuild the example programs Windows please run:

build-dotnet.cmd

from the command line.

On Linux please use the shell script:

build-mono.sh


To use the PDFjet library from Visual Studio please add reference to PDFjet
in your project:

Project -> Add Reference... -> Browse

and select PDFjet.dll


This software requires .NET SDK v2.0 or higher.

Please open the examples.html file in your browser to see 50 code examples that show how to use the library.
